FILES

index.html
style.css

music.mp3

-------------------------------------

HOW TO USE

1. Keep all files in the same folder.

2. Add your own music and rename it:

music.mp3

3. Open:

index.html

That's it!

-------------------------------------

FEATURES

✔ Animated envelope
✔ Letter slides out
✔ Typewriter animation
✔ Floating hearts
✔ Confetti
✔ Music button
✔ Mobile friendly

-------------------------------------

CUSTOMIZE THE LETTER

Open script.js

Find:

const message = `...`;

Replace it with your own message.

-------------------------------------

CHANGE THE TITLE

Open index.html

Replace

You've Got Mail 💌

with any title you like.

-------------------------------------

CHANGE THE FINAL MESSAGE

Open index.html

Find:

Thank you for being in my life.

Replace it with your own ending.

-------------------------------------

ADD PHOTOS

Create a folder named:

images

Put photos inside.

Example:

images/photo1.jpg

images/photo2.jpg

-------------------------------------

Enjoy ❤️
